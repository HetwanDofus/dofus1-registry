## Installation

```bash
npx shadcn@latest add button
```

## Usage

```tsx
import { Button } from "@/components/ui/button";
```

```tsx
<Button>Pill (default)</Button>
<Button variant="rectangle">Rectangle</Button>
<Button borderColor="white">White border</Button>
<Button variant="rectangle" borderColor="white">Rectangle white</Button>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `"pill" \| "rectangle"` | `"pill"` | Button shape variant. |
| `borderColor` | `"brown" \| "white"` | `"brown"` | Border color style. |
| `asChild` | `boolean` | `false` | Render as child element via Radix Slot. |
| `disabled` | `boolean` | `false` | Disables the button. |
