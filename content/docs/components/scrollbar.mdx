## Installation

```bash
npx shadcn@latest add scrollbar
```

## Usage

```tsx
import { Scrollbar } from "@/components/ui/scrollbar";
```

```tsx
<Scrollbar axis="vertical" className="absolute top-0 right-0 h-full" />
<Scrollbar axis="horizontal" className="absolute bottom-0 left-0 w-full" />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `axis` | `"vertical" \| "horizontal"` | `"vertical"` | Orientation of the scrollbar. |
| `className` | `string` | — | Extra classes (typically sizing and positioning). |
| `onScrollStart` | `() => void` | — | Called when the up/left arrow is clicked. |
| `onScrollEnd` | `() => void` | — | Called when the down/right arrow is clicked. |

## Sizing

The scrollbar sizes its cross-axis to `12px` (scaled by `--resolution-factor`). Its main axis is controlled by the parent container — set either `height` (vertical) or `width` (horizontal) on the parent or on `className`.
