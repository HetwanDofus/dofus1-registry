## Usage

```tsx
import {
  MainBanner,
  MainBannerChat,
  MainBannerChatInput,
  MainBannerCircle,
  MainBannerButtons,
  MainBannerIconButton,
  MainBannerMorePanel,
  MainBannerRightPanel,
  MainBannerSpells,
  MainBannerHeart,
  MainBannerFightControls,
} from "@/components/ui/main-banner";
```

```tsx
<MainBanner mode="normal">
  <MainBannerChat>{/* Chat messages */}</MainBannerChat>
  <MainBannerChatInput placeholder="Enter a message..." />
  <MainBannerCircle>{/* Minimap content */}</MainBannerCircle>
  <MainBannerHeart hp={100} />
  <MainBannerButtons>
    <MainBannerIconButton icon="stats" />
    <MainBannerIconButton icon="spells" />
    <MainBannerIconButton icon="inventory" />
    <MainBannerIconButton icon="quests" />
    <MainBannerIconButton icon="map" />
    <MainBannerIconButton icon="friends" />
    <MainBannerIconButton icon="guild" />
    <MainBannerIconButton icon="pvp" />
  </MainBannerButtons>
  <MainBannerMorePanel>
    <MainBannerIconButton icon="mount" />
    <MainBannerIconButton icon="job" />

    <MainBannerIconButton icon="title" />
  </MainBannerMorePanel>
  <MainBannerRightPanel />
  <MainBannerSpells />
</MainBanner>
```

## Components

| Component | Description |
|-----------|-------------|
| `MainBanner` | Root container. Accepts `mode="normal"` or `mode="fight"`. |
| `MainBannerChat` | Left chat message area. |
| `MainBannerChatInput` | Text input bar at bottom of chat area. |
| `MainBannerCircle` | Center circle with minimap rings. Expands on hover in normal mode. |
| `MainBannerHeart` | Heart health display with HP bar. Accepts `hp` prop (0-100). |
| `MainBannerButtons` | Action button row. Positions children at exact game coordinates. |
| `MainBannerIconButton` | Round orange button. Accepts `icon` name: `"stats"`, `"spells"`, `"inventory"`, `"quests"`, `"map"`, `"friends"`, `"guild"`, `"pvp"`, `"mount"`, `"job"`, `"achievement"`, `"event"`, `"title"`. |
| `MainBannerMorePanel` | Toggleable vertical panel above the last button slot (+/- toggle). |
| `MainBannerFightControls` | Fight-only controls. Only renders in fight mode. |
| `MainBannerRightPanel` | Notched tan panel on the right side. |
| `MainBannerSpells` | Spell bar area. |
